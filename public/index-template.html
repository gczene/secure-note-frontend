Store your notes in non-readable format!

It is a simple application for your rare used but sensitive notes.

How does it work?

Before we save your notes we translate (encrypt) it in your browser to a secure non-readable format based on a translator key given by you.
The process based on AES. You can see more here.

The key is not saved! If you forget it you can't get you data anymore!

When should you use it?
  • if you want to store sensitive information which is available anytime on the web.
  • if you use those information rare and you forget where you noted that
